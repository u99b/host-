🏭 بوت صانع البوتات - Bot Factory
=====================================

📁 هيكل المجلدات:
------------------
bot_factory.py          ← البوت الرئيسي (شغّله)
bots_templates/         ← قوالب البوتات (لا تحذفها)
created_bots/           ← يُنشأ تلقائياً عند إنشاء بوت

⚙️ الإعداد:
-----------
1. افتح bot_factory.py
2. غيّر ADMIN_TOKEN بتوكن بوت المصنع
3. غيّر ADMIN_ID بايديك على تيليغرام
4. غيّر FACTORY_TAG باسم قناتك أو معرفك
5. شغّل: python3 bot_factory.py

📦 قوالب البوتات المتاحة:
--------------------------
1. 🎨 بوت توليد الصور    (1_image_gen.py)
2. 🎰 بوت الروليت        (2_roulette.py)
3. ⬇️ بوت تحميل يوتيوب   (3_downloader.py)
4. ⚙️ بوت إدارة مجموعات  (4_group_mgmt.py)
5. 🛡 بوت حماية مجموعات  (5_group_protection.py)
6. 🛒 بوت متجر VIP       (6_vip_store.py)
7. 📞 بوت تواصل          (7_contact.py)
8. 🤖 بوت Claude AI (PHP) (8_claude_ai.php)
9. 🩸 API كتابة الدم (PHP)(9_blood_text_api.php)

-----------

- كل الحقوق محفوظة لبوت صانع
